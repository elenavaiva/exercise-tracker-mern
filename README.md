# Exercise Tracker (MERN)

Full-stack exercise tracker built with a React frontend and a Node/Express REST API backend using MongoDB. The application allows users to create, edit, and delete exercises and view them in a simple web interface.

This project was built as a portfolio project to practice full-stack development and REST API design.

## Features

- Create, edit, and delete exercises
- View all saved exercises
- RESTful API backend
- React frontend UI
- Data stored in MongoDB

## Tech Stack

- React (Vite)
- JavaScript
- HTML / CSS
- Node.js
- Express
- MongoDB (Mongoose)

## Project Structure

exercise-tracker-mern/
- backend-rest/
- frontend-react/

## How to Run Locally

Backend:

cd backend-rest  
npm install  
npm start  

You will need a `.env` file with your MongoDB connection string.

Frontend:

cd frontend-react  
npm install  
npm run dev
